# Intro

Having mongo installed localy, and create database called smsService

## Setup

```
npm install
```

## Lint

```
npm run lint
```

## Test

```
npm run test
```

## Development

```
npm run dev
```

## SeedData

To Seed data 100 data/seed

```
node seedSchedule
```

## Running Job Send SMS


```
node sendSms
```

## Running Job Checking SMS


```
node checkSms
```
